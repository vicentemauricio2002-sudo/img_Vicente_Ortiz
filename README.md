# Actividad Autonóma 1

## Descripción 
Una página web acerca de comidas típicas chilenas, en esta lista hay 3 almuerzos y 3 postres para acompañar.

## Gestión de imagenes
Con el uso de CSS en línea se le otorgó a cada imagen un atributo "width" y "height" diferentes.
Así se logró que todas las imagenes tengan un tamaño distinto.
